## 🎯 Quick Setup (5 Minutes)

### Step 1: Add Environment Variables

Go to **Render Dashboard** → **cerebrexia-app** → **Environment** tab

Add these **3 environment variables**:

#### 1. REDIS_URL
**Key:** `REDIS_URL`  
**Value:** `${cerebrexia-redis.REDIS_URL}`

*(This tells Render to use the internal Redis connection automatically)*

#### 2. RAZORPAY_KEY_ID
**Key:** `RAZORPAY_KEY_ID`  
**Value:** `rzp_test_dummy123456789`

#### 3. RAZORPAY_KEY_SECRET
**Key:** `RAZORPAY_KEY_SECRET`  
**Value:** `dummy_secret_key_abcdefghijklmnop`

### Step 2: Save and Wait

1. Click **"Save Changes"**
2. Render will automatically redeploy (2-3 minutes)
3. Watch the logs for success messages

## 🔧 Troubleshooting

### Issue 1: Redis Connection Error

**Error:** `connect ECONNREFUSED ::1:6379`

**Solution:**
1. Make sure you added `REDIS_URL` environment variable
2. Value should be: `${cerebrexia-redis.REDIS_URL}`
3. Check that **cerebrexia-redis** service is running (green status)

### Issue 2: Razorpay Error

**Error:** `'key_id' or 'oauthToken' is mandatory`

**Solution:**
1. Add `RAZORPAY_KEY_ID` = `rzp_test_dummy123456789`
2. Add `RAZORPAY_KEY_SECRET` = `dummy_secret_key_abcdefghijklmnop`

### Issue 3: Database Connection Error

**Solution:**
- Render auto-generates `DATABASE_URL`
- Make sure **cerebrexia-db** service is running (green status)
- Both services should be in the same region

## 🎯 Alternative: Manual Redis URL

If `${cerebrexia-redis.REDIS_URL}` doesn't work:

1. Go to **cerebrexia-redis** service
2. Find **"Internal Redis URL"** or **"Connection Info"**
3. Copy the URL (format: `redis://red-xxxxxxxxxxxxx:6379`)
4. Use that as the value for `REDIS_URL`
